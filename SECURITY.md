## 报告安全漏洞

**请勿通过公开 Issue 报告安全漏洞**，以避免漏洞在修复前被利用。

如果你发现了安全漏洞，请通过以下方式私下联系我们：

- 发送邮件至项目维护者（请在 README 中查看联系方式）
- 或通过 GitHub 的 [Private Security Reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) 功能提交

请在报告中包含：
1. 漏洞描述和影响范围
2. 复现步骤
3. 可能的修复建议（可选）

我们承诺在收到报告后 **7 个工作日内**给予回复，并在确认漏洞后 **30 天内**发布修复版本。
